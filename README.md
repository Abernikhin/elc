# E - language

## Variable workflow 
variable declorate

```ts
let <name>: <type>;
```

or

```
let <name>: <type> = <value>;
```

or

```
let <name> = <value>; // default auto type
```

```

```

fun <name>(<args>): <type>;
```

fun <name>;
```
